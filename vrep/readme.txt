Replace the default vrep scene "poppy_humanoid.ttt" with the file in this directory 
(or modify the default scene by creating a new Cuboid and uncheck 
"create dynamic and responsive shape")

in Windows Anaconda2:
C:\ProgramData\Anaconda2\Lib\site-packages\poppy_humanoid\vrep-scene\

in Linux Anaconda2:
/opt/anaconda2/lib/python2.7/site-packages/poppy_humanoid/vrep-scene/
